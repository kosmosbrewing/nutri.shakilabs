# 제품 설계 — 11.nutri

- 상태: Draft v0.2
- 기준일: 2026-07-10
- 서비스명: 영양만점
- Git 저장소명: `nutri.shakilabs`
- 제품 범위: 영양제 카테고리 탐색 + 일반 성인용 멀티비타민·미네랄 랭킹

## North Star

사용자가 “어떤 제품이 왜 가격 대비 효율적인가”를 공식 근거, 1일 비용, 영양소별 충족률로 3분 안에 이해하게 한다. 검색 유입을 신뢰 기반 재방문과 명확히 표시된 제휴 클릭으로 전환한다.

## 문제

- 판매가는 패키지 수량과 1일 섭취량이 달라 직접 비교하기 어렵다.
- 성분 수와 고함량 문구가 많아도 필요한 기준 충족과 비용 효율은 설명되지 않는다.
- 기존 추천 콘텐츠는 산식, 가격 시점, 출처가 불투명한 경우가 많다.
- 제품군을 섞으면 멀티비타민과 단일 영양소의 목적 차이로 순위가 왜곡된다.

## 포지셔닝

“영양제 종류를 공식 데이터로 탐색하고, 비교 근거가 완성된 제품군만 가격효율을 계산하는 데이터 도구”로 정의한다. 의료 진단, 질환 치료, 개인 처방 서비스로 표현하지 않는다.

## 대상 사용자

- 멀티비타민 첫 구매 전 가격과 구성을 비교하는 일반 성인
- 비타민D, 유산균, 오메가3 등 제품군별 공식 등록 현황을 먼저 확인하려는 사용자
- 동일 제품의 패키지와 판매처별 1일 비용을 비교하려는 사용자
- 광고성 추천보다 근거와 갱신일을 확인하려는 사용자

## 핵심 사용자 흐름

1. 카테고리 허브에서 비교하려는 영양제 종류와 공식 등록 규모를 확인한다.
2. 카테고리 페이지에서 신고번호, 제조사, 섭취 단위가 확인된 공식 레코드 예시를 살펴본다.
3. 랭킹이 제공되는 멀티비타민은 균형형 기준의 영양충족도와 1일 비용을 확인한다.
4. 예산, 제형, 1일 섭취 개수, 데이터 신뢰도로 필터링한다.
5. 최대 4개 제품의 영양소별 충족률과 가격을 비교한다.
6. 상세 페이지에서 신고번호, 라벨 근거, 가격 캡처일, 결측을 확인한다.
7. 구매를 선택하면 제휴 여부가 표시된 외부 판매처로 이동한다.

## MVP 기능

- 검증 제품 10개 랭킹
- 공식 스냅샷 기반 9개 추가 카테고리 탐색
- 카테고리별 공식 등록 레코드 예시와 분석 준비 상태
- 영양충족도와 가격효율지수 분리 표시
- 모바일 카드와 데스크톱 비교표
- 제품 상세 및 영양소별 근거
- 산정 방법, 출처, 갱신 정책 페이지
- 가격·데이터 신뢰도·주의 배지
- 링크 복사와 페이지 공유
- GA4 `ranking_view`, `filter_apply`, `compare_view`, `affiliate_click`

## 비목표

- 전 건강기능식품 통합 순위 또는 서로 다른 카테고리의 혼합 순위
- 가격·핵심 기능성 함량 근거가 없는 카테고리의 추천·효율 점수
- 질환·증상 기반 추천
- 임신·수유·소아·노인 개인화
- 복용 약물 저장과 상호작용 자동 판정
- 사용자 후기 기반 점수
- OCR 자동 승인
- 로그인과 마이페이지
- 실시간 가격 최저가 보장

## UI 원칙

- 첫 화면에서 상위 제품과 산정 기준을 바로 보여준다.
- 공식 등록 예시는 추천·순위·현재 판매 보장이 아님을 같은 화면에 표시한다.
- 카테고리마다 향후 비교 기준과 아직 필요한 근거를 공개한다.
- 점수보다 “1일 비용, 충족 영양소, 결측, 갱신일”을 먼저 설명한다.
- 모바일은 가로 스크롤 없는 카드, 데스크톱은 고정 컬럼 표를 사용한다.
- 1위는 효율 기준 1위이지 의학적 최선이 아님을 같은 화면에 표시한다.
- primary는 lime-700, 배경은 따뜻한 아이보리 계열을 사용한다.

## 수익화 원칙

- 자연 순위는 제휴 여부와 무관한 동일 산식으로 계산한다.
- 제휴 제품은 `제휴 링크` 배지와 수수료 발생 가능 문구를 표시한다.
- 광고 슬롯은 콘텐츠와 시각적으로 분리한다.
- AdSense 로더는 승인 후 추가한다.

## 성공 지표

- 출시 gate: 랭킹 가능 제품 10개, 공식 카테고리 9개, 출처 누락 0개, 산식 테스트 통과
- 품질 지표: 제품별 A/B 신뢰도, 가격 30일 이내, 잘못된 0 처리 0개
- 검색 지표: GSC 색인 성공률, 비브랜드 노출, CTR, 제품 상세 유입
- 행동 지표: 랭킹→상세, 상세→비교, 비교→외부 링크 전환율
- 신뢰 지표: 출처 열람률, 방법론 페이지 열람률, 오류 신고 건수

## 출시 조건

- 정적 HTML에서 각 페이지 title, canonical, H1, 본문이 고유하다.
- 모든 공개 제품이 `DATA_SCORING_SPEC`의 완전성 gate를 통과한다.
- 산식 golden test와 반올림·결측·가격 경계 테스트가 통과한다.
- 360px, 390px, 430px에서 overflow가 없다.
- 일반 텍스트 대비 4.5:1, 주요 터치 타깃 44px를 충족한다.
- 의료 오인 표현, 숨은 제휴, 출처 없는 순위가 없다.
